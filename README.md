# 📊 Price Intelligence Platform

Projet Data Engineering & Analytics — 2025/2026

## 📌 Description

Cette plateforme permet de :

- Collecter les prix e-commerce
- Nettoyer et transformer les données avec dbt
- Produire des statistiques analytiques
- Générer des exports CSV
- Alimenter un dashboard Streamlit

- dbt
- BigQuery
- Streamlit
- Docker
- Plotly
- Statsmodels
